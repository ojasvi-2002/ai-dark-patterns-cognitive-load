## Overview

This repository contains the experimental materials for a controlled human-subjects 
study on AI-generated dark patterns and cognitive load in cybersecurity decision-making 
— a follow-on to my Master's thesis (*When Helpfulness Becomes Harmful: How Large 
Language Models Misguide Users During Live Cybersecurity Attacks*, University of 
Luxembourg, 2026).

Where the thesis analysed LLM *outputs* against incident-response standards, this phase 
tests the downstream behavioural effect: whether cognitive load changes how likely 
people are to follow AI-generated security guidance that deviates from protocol, and 
whether that susceptibility differs between cybersecurity-trained and general IT users.

**Status:** Data collection in progress. Full results, statistical analysis, and 
theoretical discussion are part of a manuscript in preparation and are not included 
here — this repo contains only the experimental code and setup instructions needed 
to run the study procedure.
